# ASSET GUIDELINES

# Pit Wall Insight

Version: 1.0

Status: Living Document

---

# Purpose

Every visual asset inside Pit Wall Insight should feel like it belongs to the same engineering ecosystem.

This document defines the standards for selecting, creating, editing, exporting and implementing every visual resource.

Assets include

• Photography

• Illustrations

• Track Maps

• Team Logos

• Driver Images

• SVG Graphics

• Icons

• Background Textures

• Hero Graphics

• Telemetry Overlays

Consistency is mandatory.

---

# Asset Philosophy

Assets should support information.

They should never become the information.

The interface remains the hero.

Assets provide atmosphere.

---

Every asset should reinforce

Engineering

↓

Celebrity

Fan Culture

↓

# SVG Guidelines

Every SVG should

Scale perfectly.

Remain lightweight.

Use strokes consistently.

Avoid unnecessary anchor points.

Optimize before exporting.

---

# Logo Rules

Constructor logos

Small.

Supporting.

Never dominant.

Pit Wall Insight branding always takes priority.

---

# Background Assets

Backgrounds remain

Subtle.

Desaturated.

Low Contrast.

Low Opacity.

Never distract from typography.

---

# Texture Rules

# Asset Resolution

Always export at high resolution.

Retina displays are mandatory.

Never upscale low-quality assets.

---

# Optimization

Compress

Images

SVG

Videos

without visible quality loss.

Performance is part of the experience.

# Asset Quality Checklist

Before approval ask

Is it sharp?

↓

Does it match the design language?

↓

Does it reinforce engineering?

↓

Does it respect the constructor theme?

↓

Is it optimized?

↓

Is it accessible?

↓

Is it necessary?

Only then should it be included.

---

# Do

✓ Prioritize quality.

✓ Optimize everything.

✓ Maintain consistency.

✓ Reinforce engineering.

✓ Use minimal editing.

✓ Respect the visual language.

---

# Don't

✗ Stock-photo aesthetics.

✗ Cartoon graphics.

✗ Random gradients.

✗ Inconsistent editing.

✗ Low-resolution images.

✗ Decorative assets without purpose.

---

# Final Principle

Every asset should feel as though it belongs inside a Formula One engineering workstation.

If an image could just as easily belong on a social media post or a marketing website,

it probably doesn't belong in Pit Wall Insight.
